Innovative captcha with drag-n-drop verification and giving website owners a way to make money in the process. We deploy smart Ads on the captcha and ensure it wouldn't destruct the UX of your website. Before you activate Gotcha, make sure that you already have a Gotcha API Key from Gotcha by requesting to adit@gotcha.in.

== Description ==

Innovation of Captcha which no longer requires site visitors to type given words, and giving website owners a way to make money in the process. Basic function of captcha is only to prove user is human. So why you distruct them by implementing "sucks" Captcha? Dragging is quicker than typing.

Why it's important to advertise on Gotcha?
- Gotcha helps advertiser achieve their goals by delivering engaging advertising without being intrusive to our users.
- A new monetization method for your websites! Earn revenue by serving smart ads in your captcha

== Installation ==

1. Upload `gotcha-captcha.zip` to the `/wp-content/plugins/` directory
2. Go to the plugin through the 'Plugins' menu in WordPress
3. Before you activate Gotcha, make sure that you already have a Gotcha API Key from Gotcha by requesting to adit@gotcha.in.
4. Fill in all required form.
5. Click save option and Gotcha!
